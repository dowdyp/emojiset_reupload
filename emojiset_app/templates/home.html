Current Projects


Emojiset Mining

The Emojiset Mining team has created a website that can analyze emojisets from tweets and infer the meaning behind them. The code is capable of pulling tweets given various settings, and can either provide a small or large data set, which can be sent to users via email.


Epidemiology

The team has implemented a new model named Biased Majority rule model which consists of a model that resembles how opinions spread through majorities. The model contains 3 states: Adopter, Rejector and Undecided and includes a bias parameter that can be modified by the user.
